# MS KR Security Blog 소개

이 블로그는 **Microsoft Security Blog**([microsoft.com/en-us/security/blog](https://www.microsoft.com/en-us/security/blog/))의
주요 글을 한국어로 번역·정리하고, 한국 마이크로소프트 보안 팀의 솔루션별 인사이트를 전달하는 기술 블로그입니다.

## 솔루션 카테고리

| 솔루션 | 담당 |
| --- | --- |
| Purview | 병태 |
| Defender / Sentinel | 종기 |
| Entra / Intune | 상혁 |
| Security Copilot | akimcse |
| Agent 365 | akimcse · 상혁 |

> 본 사이트는 테스트 단계이며, 현재는 **Security Copilot · Agent 365** 글만 자동 게시 대상입니다.
{: .prompt-info }

## 운영 방식

- **자동 게시**: 매일 1회 Microsoft Security Blog를 크롤링하여 한국어로 번역한 뒤,
  담당자 승인을 거쳐 게시됩니다. 담당자 승인 없이는 게시되지 않습니다.
- **직접 작성**: 솔루션 담당자가 직접 작성한 글도 동일한 검수 과정을 거쳐 게시됩니다.
